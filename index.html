Técnologias:

Técnologias que eu uso no momento.

(Passe o mouse no icone de uma técnologia, para ver uma breve descrição sobre ela)

HTML5:

HTML5 (Hypertext Markup Language, versão 5) É uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML. Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade.

Fonte: Wikipedia.

Css3:

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets (ou simplesmente CSS), pela qual se define estilos para um projeto web (página de internet). Com efeitos de transição, imagem, imagem de fundo/background e outros, pode-se criar estilos únicos para seus projetos web, alterando diversos aspectos de design no layout da página.

Fonte: Wikipedia.

Python:

Python é uma linguagem de programação orientada a objetos, funcional, de tipagem dinâmica e forte.

A linguagem Prioriza a legibilidade do código sobre a velocidade ou expressividade. Combina uma sintaxe concisa e clara com os recursos poderosos de sua biblioteca padrão e por módulos e frameworks desenvolvidos por terceiros.

Fonte: Wikipedia.

Django Framework:

Django é um framework web Python de alto nível que encoraja o desenvolvimento rápido e design limpo e pragmático. Construído por desenvolvedores experientes, ele resolve grande parte do incômodo do desenvolvimento da Web, para que você possa se concentrar em escrever seu aplicativo sem precisar reinventar a roda. É gratuito e de código aberto.

Fonte: djangoproject.com

Git:

Git é um sistema de controle de versão distribuído gratuito e de código aberto projetado para lidar com tudo, desde projetos pequenos a muito grandes com velocidade e eficiência.

Fonte: git-scm.com

GitHub:

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo..

Fonte: Wikipedia.

Portifólio

No momento estou trabalhando em um back-end para o portifólio, implementando um back-end para permitir que eu modifique as fotos, textos e adicione ou remova projetos, através de uma api.
O front foi desenvolvido quase completamente em HTML5 e Css3, com exceção do menu e interação com as técnologias, que foram implementadas com JavaScrip.
O back-end está sendo desenvolvido ultilizando python com django-framework
